# AI Chatbot with Streamlit

## 📌 Project Overview
This project implements a simple AI chatbot using Python and Streamlit.  
Users can interact with the chatbot through a chat interface, and the AI generates responses using the OpenAI API.

## 🛠️ Technologies Used
- Python
- Streamlit
- OpenAI API

## ⚙️ Features
- Interactive chat interface
- Conversation history using Streamlit session state
- Integration with OpenAI language models

## 🚀 How to Run

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Set your OpenAI API Key as an environment variable

4. Run the application

streamlit run app.py

## 📚 Learning Focus
This project demonstrates how to build interactive AI-powered applications using Python and Streamlit.
